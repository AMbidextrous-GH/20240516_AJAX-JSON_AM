＜リポジトリ名＞

　20240516_AJAX-JSON_AM

＜概要＞

　自分の好きなこと(配信)を紹介するために作成

＜目的＞

　・好きなこと(配信)を共有し、同じ趣味を持つ人々と交流すること
　・好きなこと(配信)についての詳細な情報を提供すること
　・訪問者に新しい興味や趣味を見つける手助けをすること
　

＜サイト構成＞

　・news(更新情報)
　　　更新情報を過去3つまで表示
　
　・about(サイトについて)
　　　このサイトを作成するに至るまでの経緯

　・words(用語紹介)
　　　配信界隈で使われている用語の紹介

　・favorites(好きな配信者)
　　　好きな配信者の紹介及び各チャンネルへのリンク

　・contact(お問い合わせ)
　　　質問やフィードバックを送信するフォーム

　・comments(一言コメント)
　　　サイトへのコメントを送信するフォーム

＜使用技術＞

　HTML,CSS,javaScript,PHP,MySQL
